## Build Setup

``` bash
# install dependencies
npm install

# build electron application for production
npm run build
```

## Feature

* Json format, Base64 md5 uri encode/decode
* RGB HEX CSS transform, Color constant, Color picker
* Image to base64, Base64 to image
* Regex test
* Host edit
* System info
